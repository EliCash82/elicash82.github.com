Projects

    CorporaDB

    CorporaDB is a living database of 19th Century lit. It is deployed using Ruby/Sinatra. It utilizes SASS for styling, SLIM for markup, with PostgreSQL as the underlying relational database.

    Anderson's HyperLit Engine (AHLE)

    AHLE is a simple classic adventure game template for use in collegiate level and high school level as a way to present computer coding (specifically Ruby) in a traditional English/Lit style classroom setting.

    The Gentle Introduction Resource

    The Gentle Introduction Resource was the foundation of my thesis. It provides Digital Humanities classrooms with a private social data stream to crowdsource relevant research material. It was built with Ruby on Rails.
